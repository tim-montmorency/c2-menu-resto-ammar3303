## Description

Le menu est divisé en quatre sections thématiques d'halloween, chacune proposant trois choix :

Cocktails : Des boissons originales aux noms sombres (Venin, Seringue, Cyclope).

Entrées : Des amuse-bouches visuels (Soupe de sorcière, Œufs de monstres, Doigts de momies).

Plats Principaux : Des options consistantes mais effreyant .

Desserts : Des créations sucrées sanglantes (Cerveau éclaté, Pomme empoisonnée, Cœur en chocolat).

**Décisions CSS notables :**

Mise en page flexible : J'ai organisé le menu pour qu'il s'adapte automatiquement à la taille de l'écran. Cela permet de garder les noms des plats et les prix bien alignés, peu importe la longueur du texte.

Ambiance visuelle : J'ai choisi des couleurs sombres avec des touches de rouge et d'orange pour rappeler l'Halloween. J'ai aussi ajouté des effets d'ombre pour donner un aspect mystérieux et "flou" aux textes.

Choix des écritures : J'ai utilisé des polices spéciales qui font peur pour les titres, tout en gardant une écriture simple pour les descriptions afin que le menu reste facile à lire.

## Difficultés rencontrées

Affichage sur mobile : Le plus dur a été de faire en sorte que le menu reste joli sur un téléphone. Il fallait passer d'une présentation large à une présentation verticale sans gâcher le style.

Lisibilité : Il a fallu trouver le bon dosage pour que le site ait l'air effrayant sans que les effets visuels n'empêchent de lire correctement les plats.
